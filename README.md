# Eyeball Factory — Nancy Role Applications

Hiring quiz site for Nancy creative roles:
- **VSL Editor** — Direct response video sales letter editing
- **Clipping Editor** — Social media short-form clip editing

## Live URL
https://eyeballquiz-tc9qtxtj.manus.space/

## Routes
- `/` — Home (role overview)
- `/vsl-editor` — VSL Editor application
- `/clipping-editor` — Clipping Editor application

## Stack
- React + TypeScript (Vite build)
- Fonts: Fraunces (display) + Nunito Sans (body)

## Deployment
Static site — deploy `index.html` + `assets/` to any static host (Vercel, Netlify, Cloudflare Pages).

For SPA routing, configure a catch-all redirect to `index.html`.
